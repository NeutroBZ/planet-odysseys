## Image Requirements

### Dimensions
- **Tour Images**: 1200x900px (4:3 ratio)
- **Destination Images**: 1200x1500px (4:5 ratio for vertical cards)
- **Blog Images**: 1600x1000px (16:10 ratio)
- **OG Image**: 1200x630px

### Format
- Use WebP for best performance
- JPG as fallback
- Optimize all images (compress to <200KB)

### Naming Convention
- Use kebab-case (lowercase with hyphens)
- Descriptive names
- No spaces or special characters

## Where to Get Images

You can download the original images from:
1. **Planet Odysseys Website**: https://planetodysseys.com
2. **WordPress Media Library**: If you have access
3. **Stock Photos**: Use high-quality Peru/Lake Titicaca images

## Optimization Tools

- **TinyPNG**: https://tinypng.com
- **Squoosh**: https://squoosh.app
- **ImageOptim**: https://imageoptim.com

## Quick Start

1. Download images from the original website
2. Resize to recommended dimensions
3. Optimize file sizes
4. Place in appropriate directories
5. Ensure filenames match those in data files
